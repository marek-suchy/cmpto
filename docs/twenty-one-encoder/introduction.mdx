# Introduction

This document supplies basic information on how to operate Comprimato Twenty-One Encoder. Inside the document, you will find information about installation, monitoring, UI operation, and valuable tools.

## Initial Setup

After unboxing the Twenty-One Encoder appliance, please use the following steps to configure:

Insert SFP28 modules, please find list of known compatible SFP28 modules in section 1.1

Ensure link with 10 Gbps copper NIC (For management and data)

Connect video monitor to display port 1 (right side)

Connect keyboard and mouse

Twenty-One Encoder supports IS-04 Distributed Registry, the default NIC for this is LAN 1 (eno1np0)

The Twenty-One Encoder UI should be available on the GPU’s display port. Username: admin, Password: admin

Click setup on the UI to run through the initial configuration dialogue:
