Support requests can be raised by emailing [support@comprimato.com](mailto:support@comprimato.com)

1. Approximate or exact Date/Time/Time Zone of the issue encountered
2. Which product? i.e. Bridge Live, Live Transcoder?
3. What happened and to which host (name or IP) and pipeline ID?
4. Diagnostics package, available on the transcoder web UI, please upload to [www.download.comprimato.com](http://www.download.comprimato.com) if possible, contact Comprimato Support for access
5. Screengrab of the UI if possible or any messages seen
6. Input/Output captures
