## OPEX

1. **Hourly** — Per pipeline, per hour
2. **Monthly** — Per maximum number of pipelines used concurrently within a month
3. **Annual** — Per year, limited number of pipelines set, must be pre-paid

## CAPEX

1. **Perpetual license** — Available with on-prem Live Transcoder
