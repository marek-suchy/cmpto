# Login to Web UI

After you finish the installation, you can log in to Live Transcoder web UI. It's available through web browsers (desktop version is recommended) on the host machine's public IP address.

![](/img/cloud-deployment/ui-address-bar.png)

The default web UI password is `admin`. You can change it later.

![](/img/cloud-deployment/ui-login.png)

We recommend switching to HTTPS on the public Internet. Browsers will show you a warning on an invalid HTTPS certificate. This is OK; you can proceed.

![](/img/cloud-deployment/ui-https-exception.png)

# Change UI password

We strongly recommend changing the UI password. Change it in Admin / Security.

![](/img/cloud-deployment/ui-password-change.png)
