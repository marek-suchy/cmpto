Live Transcoder is a subscription-based software distributed as a Docker image.

Please get in touch with our Support at [support@comprimato.com](mailto:support@comprimato.com) - our specialists will prepare your subscription and send you access to the [Customer Dashboard](https://transcoder.comprimato.com/) page.

![Live Transcoder Dashboard](/img/cloud-deployment/dashboard.png)

*Customer Dashboard*

On the Customer Dashboard, you'll find the following:

1. Personal access token to our private DockerHub repository\
   It looks similar to this: `dckr_pat_XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX`. Older tokens can look like this `xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx`.
2. docker-compose file (`docker-compose.yml`) with a prefilled Live Transcoder license key.
3. Live Transcoder license key\
   It looks similar to this: `64544bd5-8df6-490c-bce7-468d035a8b1a`

In the meantime, you can start preparing the instance and OS for the Live Transcoder Docker container in your preferred cloud provider or on-premise. When you got your subscription ready, you can run the Docker container.
