# Twenty-One Encoder

<p className="resource-spec__meta">Software-based ST 2110 encoder and decoder appliance for live broadcast production</p>

## Hardware Specifications

| Parameter | Value |
|---|---|
| Form factor | 1U Rackmount |
| Network connectivity | 2× 10GbE RJ45; 2× 10/25GbE SFP28 |
| Management | IPMI 2.0 with virtual media over LAN and KVM-over-LAN |
| Power supply | 100–240 VAC, 50/60 Hz, redundant — 320 W typical |
| Dimensions (W × H × D) | 17.2" × 1.7" × 18.8" (437 mm × 43 mm × 477 mm) |
| Package (W × H × D) | 27" × 8" × 24" (686 mm × 203 mm × 610 mm) |
| Net weight | 16 lbs (7.3 kg) |
| Gross weight | 25.8 lbs (11.7 kg) |
| Operating temperature | 10°C to 35°C (50°F to 95°F) |
| Non-operating temperature | −40°C to 70°C (−40°F to 158°F) |
| Operating humidity | 8% to 90% (non-condensing) |
| Non-operating humidity | 5% to 95% (non-condensing) |
