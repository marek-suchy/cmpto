# Bridge Live

<p className="resource-spec__meta">1RU broadcast-quality, low-latency turnkey system for REMI, contribution, and streaming — developed by AJA and Comprimato</p>

## SDI Video Formats

### Bridge Live 12G-4

| Format | Frame rates |
|---|---|
| UltraHD 3840×2160p | 23.98, 24, 25, 29.97, 30, 50, 59.94, 60 |
| 2K 2048×1080p | 23.98, 24, 25, 29.97, 30, 47.95, 48, 50, 59.94, 60 |
| HD 1080p | 23.98, 24, 25, 29.97, 30, 50, 59.94, 60 |
| HD 1080i | 50, 59.94, 60 |
| HD 720p | 50, 59.94, 60 |
| SD 625i | 50 |
| SD 525i | 59.94 |

SDI standards: 12G-SDI (SMPTE ST 2082), 6G-SDI* (SMPTE ST 2081), 3G-SDI (SMPTE ST 424), 1.5G-SDI (SMPTE ST 292M), 270 Mbit/s SDI (SMPTE ST 259M-C) — all 10-bit.

*Only UltraHD 2160-line single link supported.

### Bridge Live 3G-8

| Format | Frame rates |
|---|---|
| HD 1080p | 23.98, 24, 25, 29.97, 30, 50, 59.94, 60 |
| HD 1080i | 50, 59.94, 60 |
| HD 720p | 50, 59.94, 60 |
| SD 625i | 50 |
| SD 525i | 59.94 |

SDI standards: 3G-SDI (SMPTE ST 424), 1.5G-SDI (SMPTE ST 292M), 270 Mbit/s SDI (SMPTE ST 259M-C) — all 10-bit.

## Video Color Component Sampling

| Source | Sampling |
|---|---|
| SDI | YCbCr, 4:2:2, 10-bit |

## Audio I/O

| | Details |
|---|---|
| Audio input (digital) | 16-channel, 16 and 24-bit SDI embedded audio, 48 kHz, synchronous per SDI connection |
| Audio output (digital) | 16-channel, 16 and 24-bit SDI embedded audio, 48 kHz, synchronous per SDI connection |

## Metadata

- MPEG-2 ancillary packet support (SMPTE ST 2038)
- HDR over SDI end-to-end via VPID signaling (SDR/HDR Transfer Characteristics, Colorimetry, Luminance)
- Closed Captioning (EIA-608/708), Line 21 Captions
- Metadata pass-through
- SCTE-35, SCTE-104
- Subtitles embedded within H.264 SEI messages
- Timecode embedded within HEVC and H.264 SEI messages
- Electronic program guide (EPG) on input

## Video Processing

- Auto Color Component Sampling based on I/O settings
- Cropping / Padding
- Color adjustments
- De-interlacing
- Frame rate 50% down-conversion (60p→30p, 59.94→29.97, 50p→25p)
- Frame rate up-conversion (50i→50p)
- Logo insertion
- Resizing

## Audio Processing

- Channel remapping
- Gain
- Sample rate conversion
- Up/down-mixing (e.g., 5.1 to stereo and vice versa)

## Synchronization

| Parameter | Options |

| Decoder synchronous multi-channel transport | — |

## Configuration & Monitoring

- Remote access via browser
- REST API
- SNMP
