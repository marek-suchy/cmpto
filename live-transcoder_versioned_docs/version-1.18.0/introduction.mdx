# 1.0 Introduction

This document supplies basic information on how to operate the Comprimato Live Transcoder Appliance. Inside the document, you will find information about installation, operation, and troubleshooting.

Within the packaging a Quick Start Guide is available. This contains setup instructions, credentials for UI, SSH and IPMI and also the serial number of the appliance.If this is missing please contact support@comprimato.com and an electronic copy can be sent to you.

*Figure: Live Transcoder APPLIANCE - BACK*

Rack Mount the Encoder

Secure the encoder in your server rack using the provided rack rails. Ensure it is properly aligned and fastened.

Connect Power and Networking Cables

Plug in the dual power cables for redundancy.

Connect the RJ-45 10GbE cables for network connectivity and stream data.

Optionally, connect a USB keyboard, mouse, and monitor via the DisplayPort 1 interface for local access.

Power On the Appliance

Press the power button to start the appliance. Wait for it to boot.

Access the Graphical User Interface (GUI) through one of the following methods:

Directly on the monitor connected via DisplayPort 1.

Through the network on dynamically assigned IP address if connected to LAN Port 1 (check your network’s DHCP server to find the IP).

Through the network on the pre-configured static IP address 192.168.1.42 if connected to LAN Port 2.

Login and Accept the EULA

Log in using the credentials provided down below.

Read and accept the End User License Agreement (EULA) to proceed.

Initial Setup Dialogue

The Setup Dialogue will start automatically. Follow the prompts.

Enter the hostname for your appliance.

Configure the network ports and assign IP addresses.

Ensure that at least one of the 10GbE network interfaces is configured for administration and management purposes.

Verify the settings to avoid any IP address conflicts within your network.

Finalize and Save Settings

Save your configuration.

Congratulations

Your Live Transcoder is now ready for usage, and you can proceed to stream setup.

Refer to the user manual for advanced setup options or troubleshooting if needed.

## 1.1 Default UI login

Each Appliance has a unique username and password that are available on the quick-start guide found in the packaging of the Live Transcoder Appliance. If this is missing please contact support@comprimato.com

If a reinstallation has been performed, the default credentials are as follows:UI User: admin Password: adminSSH User: transcoder Password: transcoder
