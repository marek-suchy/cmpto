When contacting Comprimato Support, include the following information to ensure your issue is resolved as quickly as possible.

## Contact

Send support requests by email to the Comprimato support team. See [Contact Details](../advanced/contact-details) for the email address.

## Required Information

Include all of the following in your support request:

1. **Timing details** — The approximate or exact date, time, and time zone when the issue occurred.

2. **Product identification** — Which Comprimato product is affected (e.g., Bridge Live, Live Transcoder).

3. **Incident description** — A clear description of what occurred, including:
   - The affected host instance identifier
   - The pipeline ID (if applicable)

4. **Diagnostics package** — Download the diagnostics package from the web UI at **Setup > Diagnostics & Support** and upload it to the Comprimato download server. Contact Comprimato Support to obtain upload credentials for [www.download.comprimato.com](https://www.download.comprimato.com).

5. **Screenshots** — UI screenshots or error messages that capture the issue.

6. **Media samples** — Input and output video captures, when relevant to the problem.

## Diagnostics Package

The diagnostics package contains logs and system information for the past 7 days. It is the most important artifact for troubleshooting. See [Diagnostics Package (Logs)](../advanced/diagnostics) for details on what it contains and how to interpret it.
