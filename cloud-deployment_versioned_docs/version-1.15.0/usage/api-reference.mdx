Every Live Transcoder instance includes a built-in API reference that matches the version currently deployed.

## Accessing the API Reference

The API reference is available directly in the Live Transcoder web UI:

1. Log in to the web UI.
2. Click the **API** button in the top navigation menu.

The API reference is version-specific — it always reflects the exact endpoints and parameters available in your installed version.

## For AI Agents and Automation

## Documentation Structure

The full documentation is organized into five main sections:

1. **Prerequisites** — Subscription requirements
2. **Installation** — Cloud deployment (AWS, Google Cloud, Oracle Cloud, Azure) and on-premises
3. **Run It** — Docker container deployment with GPU and CPU-only options
4. **Usage** — Getting started, upgrade, and downgrade procedures
5. **Advanced Support** — Troubleshooting, diagnostics, and performance testing

## Version History

The documentation covers versions v1.14.0 through v1.18.0, with version-specific content available via the version selector.
