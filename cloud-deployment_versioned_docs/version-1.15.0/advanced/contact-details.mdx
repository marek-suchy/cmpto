## Before Contacting Support

When raising a support request, prepare the following information:

- Approximate date, time, and time zone of the issue
- Which product is affected (e.g., Live Transcoder, Bridge Live)
- Description of the issue, including host instance ID and pipeline ID
- Diagnostics package from **Setup > Diagnostics & Support** in the web UI
- Screenshots of any error messages

See [Raising Support Requests](../troubleshooting/support-requests) for the full checklist.
